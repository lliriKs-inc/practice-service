# Student guide

1. Open the cohort invitation, register or sign in, complete the questionnaire and select a practice track.
2. Submit the application and, when available, upload the test task file. Replacing a submission replaces the previous file.
3. After approval, open the weekly practice calendar. Save a description and any links for each working day; saving again replaces that day's links.
4. Fill the student-owned document fields and check document readiness.
5. Upload the report as PDF, DOC or DOCX. Replacing it sends the new version back for administrator review.
6. After the required fields and approvals are complete, generate and download the available DOCX documents.

Reports and generated files are private. Use only the download buttons in the application, do not reuse a link from another account, and contact the administrator if an expected practice day or document is missing.
